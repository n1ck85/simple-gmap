1.  Install or Upload the plugin via FTP.
2.  Place the shortcode on any page, the minimun attribute the shortcode needs is [lat], [lng] & [key] which is the Google maps API key.
    The map will inherit the container width and the deafult height is 400px, see sample shortcode below for guidance.
	
    [simple-gmap lat="(required)" lng="(required)" icon="" height="" key="(required)"]
